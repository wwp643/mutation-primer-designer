# mutation-primer-designer
一款用于快速设计氨基酸单点突变实验所需上下游引物的网页工具。
核心功能
输入野生型基因序列和突变位点（格式如 A10V），自动生成含突变位点的引物序列
直观高亮显示引物中的突变碱基（野生型碱基红色，突变型碱基绿色）
计算并展示引物的 Tm 值、GC 含量等关键参数
支持导出 CSV 格式（可直接用 Excel 打开，无乱码）和 FASTA 格式（引物名称以 -F/-R 区分，方便实验使用）
适用场景
分子生物学实验中，针对蛋白质氨基酸单点突变的引物设计，帮助科研人员快速获取符合实验要求的引物序列，减少手动设计的繁琐与误差。
使用方法
输入野生型基因序列（仅含 A/T/C/G）
填写突变位点（每行一个，格式为 “野生型氨基酸 + 位置 + 突变后氨基酸”，如 E143C）
可调整引物长度、Tm 值、GC 含量等参数
点击 “生成引物” 查看结果，如需保存可导出为 CSV 或 FASTA 格式
